# devestation Configuration Skeleton

This directory contains configuration files for the Hyprland-based workstation setup.

## Directory Structure:
- **@sys/** - Files copied to system root (/)
- **@user/** - Files copied to user home directory
- **@root/** - Files copied to root home directory

## Usage:
Place configuration files in the appropriate directories above.
They will be automatically copied during devestation installation.

## Notes:
This skeleton is used by the devestation.sh configuration.
It provides a Hyprland desktop environment with NVIDIA LTS drivers.
